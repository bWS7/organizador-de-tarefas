# Gerenciador de Tarefas por Categoria

Aplicação React usando Context API para gerenciar categorias e tarefas.

## Como rodar

1. Certifique-se de ter o **Node.js** instalado (versão 18 ou superior).
2. Dentro da pasta do projeto, rode:

```bash
npm install
npm run dev
```

3. Abra o navegador no endereço indicado no terminal (geralmente `http://localhost:5173`).
